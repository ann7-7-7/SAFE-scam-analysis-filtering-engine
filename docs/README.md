## Project Overview
S.A.F.E. (Scam Analysis & Filtering Engine) is a web-based AI system designed to analyse suspicious digital messages (SMS, email, WhatsApp) and provide:

- Scam likelihood scores
- Explainable highlighting of risky phrases
- Human-readable safety guidance
- Privacy-first processing with no raw message storage

This repository contains design artefacts, documentation, ethics materials, project planning evidence, and the early proof-of-concept structure for the system.
