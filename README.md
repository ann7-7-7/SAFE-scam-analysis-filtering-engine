## Project Overview

S.A.F.E. (Scam Analysis & Filtering Engine) is a web-based artificial intelligence system designed to analyse suspicious digital messages (SMS, email, WhatsApp) and provide:

- Scam likelihood scores
- Explainable highlighting of risky phrases
- Human-readable safety guidance
- Privacy-first processing with no raw message storage

The project focuses on combining NLP-based scam detection with explainable AI and GDPR-aligned system design.
